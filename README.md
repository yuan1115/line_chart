#### 折线图管理
	
	基于matplotlib库的折线图绘制

#### 环境
	
* python3.6
* wxpython
* pylab
* matplotlib

#### 目录结构

	|-- __pycache__      执行缓存
	|-- dist             打包好的程序
	|-- gui.py           gui
	|-- i.ico            图标
	|-- images.py        图标转成的python
	|-- yun.py           主程序
	|-- README.md        说明文档
